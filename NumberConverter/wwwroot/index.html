How to Use

Send a POST request to /api/convert with JSON:

{
  "number": "1010",
  "fromBase": 2,
  "toBase": 16
}

Example response:

{
  "original": "1010",
  "fromBase": 2,
  "toBase": 16,
  "converted": "A"
}

Or test it interactively using the form above, or via Swagger UI.
