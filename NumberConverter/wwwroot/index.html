About Number Bases & Base Conversion

A number base (radix) is the number of unique digits used to represent numbers. Bases are widely used in computing, mathematics, and programming.

  • Binary (Base 2): Uses 0 and 1. Fundamental in computing and electronics.
  • Decimal (Base 10): Standard human-readable numbers (0–9).
  • Hexadecimal (Base 16): Uses 0–9 and A–F. Common in programming, memory addresses, and color codes.
  • Octal (Base 8): Uses 0–7. Sometimes used in legacy computing systems.

Base converters allow developers and students to quickly translate numbers from one system to another.
